## Project description

This program works as a digital shopping cart, with the abilty to add and remove items with the ability to calculate total cost.

## How to compile and run the program

How to compile (if applicable) and run the project.

```Windows PowerShell
open a terminal and run "javac *.java"
then run "jar cfe ShoppingCart.jar ShoppingCartManager *.class"
then you can run the program with "java -jar ShoppingCart.jar"
```

## UI Design

The program will first ask for your name and date, then it will output the menu in which you can add, remove, change quantity, output descriptions, and output shopping cart (Fig 1). After completing your shopping cart, you can then ask for the total price, and then hit quit to stop the program(Fig 2).
